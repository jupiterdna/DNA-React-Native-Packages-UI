# Avatar

Avatars are prevalent in UI design, appearing in various contexts such as lists and profile screens. They serve as visual representations of users and can incorporate photos, icons, or text to convey identity or information.

## Usage

<div class="row inline-flex-center items-center">
  <div class="col col--3">
    <h4>Import</h4>
  </div>
  <div class="col col--9">
    <CodeBlock>{`import { DNAAvatar } from '@rndna/avatar';`}</CodeBlock>
  </div>
</div>

## Props

<div class='table-responsive'>

| Name                    | Type                                                       | Default                                           | Description                                                                             |
| ----------------------- | ---------------------------------------------------------- | ------------------------------------------------- | ----------------------------------------------------------------------------------------|
| `testID`                | String                                                     | `Undefined`                                       | Property to be used for testing                                                         |
| `name`                  | String                                                     | `Undefined`                                       | Property that defines the full name of the avatar.                                      |
| `imageSource`           | string / ImageSrc                                          | `undefined`                                       | Property that defines the source URL of the image/element to be used.                   |
| `size`                  | SizeProps `( "xs","sm","md","lg", "xl")`                   | `md`                                              | Property that defines the size of the component.                                        |
| `borderRadius`          | `("squared", "soft_edged", "rounded")`                     | `undefined`                                       | Property that defines the border dimensions of the component.                           |
| `color`                 | `("primary", "secondary","info", "success", "warning", "danger", "default")`       | `default`                 | Property that changes theme of the component, i.e. background and text color,           |
| `alt`                   | String                                                     | `undefined`                                       | Property that defines the alternative text of the image in the avatar.                  |
| `icon`                  | React.JSX.Element                                          | `undefined`                                       | Property that enables user to select icon from icon library.                            |
| `onPress`               | `(event: GestureResponderEvent) => void`                   | `undefined`                                       | Handler when the component is clicked.                                                  |
</div>

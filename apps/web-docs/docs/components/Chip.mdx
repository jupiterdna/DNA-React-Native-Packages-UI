# Chip

Chips are condensed components capable of representing inputs, attributes, or actions.

## Usage

<div class="row inline-flex-center items-center">
  <div class="col col--3">
    <h4>Import</h4>
  </div>
  <div class="col col--9">
    <CodeBlock>{`import { DNAChip } from '@rndna/chip';`}</CodeBlock>
  </div>
</div>

## Props


<div class='table-responsive'>

| Name                    | Type                                                       | Default                                           | Description                                                                                                                                                  |
| ----------------------- | ---------------------------------------------------------- | ------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `label`                 | string                                                     | `Chip`                                            | Where you can define the label of the component.                                                                                                             |
| `icon`                  | `React.JSX.Element`                                        | `undefined`                                       | Property that enables user to select icon from icon library                                                                                                  |
| `variant`               | `solid`, `outlined`, `soft`                                | `solid`                                           | Changes the variation of the element.                                                                                                                        |
| `size`                  | `xs`, `sm`, `md`, `lg`, `xl`                               | `md`                                              | Property that defines the size of the component or a selection                                                                                               |
| `isClosable`            | boolean                                                    | `false`                                           | Property that places a Closable button as X icon on the component to make it closable                                                                        |
| `isDisabled`            | boolean                                                    | `false`                                           | Property where the user can enable or disable the certain attribute or field                                                                                 | 
| `color`                 | `primary`, `secondary`, `info`, `success`, `warning`, `danger` | `primary`                                     | Property that changes theme/state of the selection i.e. background and text color                                                                            |              
| `borderRadius`          | `rounded`, `soft_edged`                                    | `rounded`                                         | Defines the border dimensions of the component                                                                                                               |
| `onPressClose`          | `(event: GestureResponderEvent) => void`                   | `() => {}`                                        | Handler when the component is about to close                                                                                                                |
| `style`                 | `ViewStyle`                                                | `undefined`                                       | Style that is an optional `StyleProp<ViewStyle>` object that applies to the outer content View component                                                                                                                      |    

</div>

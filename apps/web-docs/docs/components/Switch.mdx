# Switch

Switch is a component that allows the user to toggle between two states: on and off.

## Usage

<div class="row inline-flex-center items-center">
  <div class="col col--3">
    <h4>Import</h4>
  </div>
  <div class="col col--9">
    <CodeBlock>{`import { DNASwitch } from '@rndna/switch';`}</CodeBlock>
  </div>
</div>

## Props


<div class='table-responsive'>

| Name                    | Type                                                       | Default                                           | Description                                                                                                                                                  |
| ----------------------- | ---------------------------------------------------------- | ------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `title`                 | string                                                     | `undefined`                                       | Property that defines the title of the component.                                                                                                            |
| `titlePosition`         | `top`, `bottom`, `left`, `right`                           | `top`                                             | Property that defines the position of title.                                                                                                                 |
| `labelRight`            | string                                                     | `undefined`                                       | Property that defines the right label of the component.                                                                                                      |
| `labelLeft`             | string                                                     | `undefined`                                       | Property that defines the left label of the component.                                                                                                       |
| `size`                  | `xs`, `sm`, `md`, `lg`, `xl`                               | `undefined`                                       | Property that defines the size of the component or a selection                                                                                               |
| `color`                 | `primary`, `secondary`, `info`, `success`, `warning`, `danger`, `default` | `primary`                          | Property that changes theme/state of the selection i.e. background and text color                                                                            |                                                             
| `disabled`              | boolean                                                    | `false`                                           | Property that defines the disabled state of the component.                                                                                                   | 
| `value`                 | boolean                                                    | `undefined`                                       | Property that defines the value of the component.                                                                                                            |
| `onValueChange`         | `(value: boolean) => void`                                 | `undefined`                                       | Property that defines the callback function when the value of the component is changed.                                                                      |
| `iconLeft`              | ` React.JSX.Element`                                       | `undefined`                                       | Property that defines the left icon of the component.                                                                                                        |
| `iconRight`             | ` React.JSX.Element`                                       | `undefined`                                       | Property that defines the right icon of the component.                                                                                                       |
| `variant`               | `solid`, `filled`                                          | `solid`                                           | Property that defines the variant of the component.                                                                                                          |

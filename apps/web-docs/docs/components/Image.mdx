# Image

Image component for various image types: network, static, local, camera roll.

## Usage

<div class="row inline-flex-center items-center">
  <div class="col col--3">
    <h4>Import</h4>
  </div>
  <div class="col col--9">
    <CodeBlock>{`import { DNAImage } from '@rndna/image';`}</CodeBlock>
  </div>
</div>

## Props


<div class='table-responsive'>

| Name                    | Type                                                       | Default                                           | Description                                                                                                                                                  |
| ----------------------- | ---------------------------------------------------------- | ------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `ratio`                 | `1:1`, `3:2`, `4:3`, `8:5`, `16:9`                         | `16:9`                                            | Property that allows you to define the desired width-to-height ratio of the image.                                                                          |
| `fit`                   | `cover`, `contain`                                         | `cover`                                           | Property that allows you to set the size to fit the available space of the image wrapper                                                                    |
| `src`                   | `string`                                                   | `undefined`                                       | Property that defines the source URL of the image/element to be used.                                                                                       |
| `size`                  | `xs`, `sm`, `md`, `lg`, `xl`                               | `md`                                              | Property that defines the size of the component or a selection                                                                                              |
| `borderRadius`          | `squared`, `soft_edged`, `rounded`                         | `soft_edged`                                      | Defines the border dimensions of the component                                                                                                              |

</div>

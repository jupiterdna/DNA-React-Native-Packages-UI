# Icon

Icon components for displaying small graphical representations of programs or files from various sources: network, static, local, camera roll.

## Usage

<div class="row inline-flex-center items-center">
  <div class="col col--3">
    <h4>Import</h4>
  </div>
  <div class="col col--9">
    <CodeBlock>{`import { UserIcon } from '@rndna/icon';`}</CodeBlock>
  </div>
</div>

## Props


<div class='table-responsive'>

| Name                    | Type                                                       | Default                                           | Description                                                                                                                                                  |
| ----------------------- | ---------------------------------------------------------- | ------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `width`                 | `string`, `number`                                         | `24`                                              | Property that defines the width of the component.                                                                                                            |
| `height`                | `string`, `number`                                         | `24`                                              | Property that defines the height of the component.                                                                                                           |
| `viewBox`               | string                                                     | `0 0 24 24`                                       | Property that defines the position and dimension, in user space, of an SVG viewport                                                                                                         |
| `preserveAspectRatio`   | string                                                     | `xMidYMid meet`                                   | Property that defines how the aspect ratio of the viewBox is preserved.                                                                                      |
| `color`                 | string                                                     | `#000`                                            | Property that defines the color of the component.                                                                                                            |
| `title`                 | string                                                     | `undefined`                                       | Property that defines the title of the component.                                                                                                            |
| `size`                  | `string`, `number`                                         | `24`                                              | Property that defines the size of the component.                                                                                                             |

</div>
